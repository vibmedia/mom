## Rules for Each File
1. **Image prompt** at the top of each slide section (for AI generation)
2. **ALL data** from source document included — tables, numbers, lists, everything
3. **No summarizing** — if the source has 12 months of detail, all 12 months go in
4. **Slide breaks** marked with `---` horizontal rules
5. **Speaker notes** included below each slide's data
