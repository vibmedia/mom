**Market Segments:**
| Segment | Size | Key Players | Our Play |
|---------|------|-------------|----------|

6. NRI nostalgia (overseas Indians craving "ghar ka swaad")

**Our Target:** 0.002% of ₹25K Cr = ₹50Cr (Year 3 goal)

**Jhaji Store — The Pioneer**

| Metric | Value |

| Category | Pickles, Chutneys, Murabba |

**Growth Journey:**
- 2020: Started selling on WhatsApp, 50 orders/month

- 2022: Went viral on Instagram (village cooking videos), Shark Tank application
- 2023: ₹85L funding from Inflection Point Ventures

- Jhaji = village collective → MKN = one mother (stronger personal brand)
- Jhaji = Bihar only → MKN = Himachal + eventually all states (Maa Ka Network)
- Jhaji weak on: brand consistency, packaging quality, website experience
- MKN advantage: tech founder (better digital ops), content creator co-founder

## SLIDE 3: FARMDIDI — COMPETITOR DEEP DIVE

### Prompt
```
Design a 16:9 slide titled "Competitor Study: FarmDidi". Same layout as Jhaji slide. Left: metrics card. Right: growth timeline. Bottom: learnings. Cream bg.
```

### Data

**FarmDidi — The Network Model**

| Metric | Value |
|--------|-------|
| Founded | 2021 |
| Location | Multi-state (Rural women network) |
| Category | Pickles, Papads, Spices, Dry Fruits |
| Revenue (est.) | ₹18Cr ARR |
| Funding | ₹7Cr seed round |
| Channels | D2C website, Amazon, Flipkart, Quick-commerce |
| Instagram | 200K+ followers |
| YouTube | Active |
| USP | Network of rural "didis" across India making products |

**The FarmDidi Model:**
- NOT one kitchen — it's a platform connecting rural women makers to urban buyers
- Each "didi" is a real woman in a village
- Products made across multiple states
- FarmDidi handles: branding, packaging, marketing, logistics
- Didis handle: production

**What We Learn:**
1. Multi-state model works → validates our "Maa Ka Network" Phase 2 strategy
2. ₹7Cr seed → ₹18Cr ARR shows investor appetite for this category
3. "Story of the maker" is the KEY selling point
4. Platform model scales faster but has QUALITY CONTROL challenges
5. Their packaging is inconsistent → our standardized glass jars win

**How We Differentiate:**
- FarmDidi = platform (many makers, less control) → MKN = single brand (Anjula ji core + curated network)
- FarmDidi struggles with: batch consistency, quality control across villages
- MKN advantage: one farm, one kitchen, one QC person (Anjula ji) → consistent quality
- When we expand (Phase 2), we TRAIN mothers on our standards, not just buy their output
