# D2C Packaging Design

> Design packaging that preserves quality, attracts buyers, and complies with FSSAI.

## Core Framework

### Packaging Format Decision

| Format | Best For | Cost (per unit) | Shelf Life | Premium Feel |
|--------|----------|----------------|-----------|-------------|
| **Stand-up Pouch** | Namkeen, masala, dry items | ₹3-8 | Good | Medium |
| **Glass Jar** | Pickles, chutneys, sauces | ₹15-40 | Excellent | High |
| **PET Jar** | Pickles (budget), snacks | ₹8-15 | Good | Medium |
| **Paper Box** | Gift sets, dry mixes | ₹10-25 | Basic | High |
| **Tin Can** | Sweets, premium items | ₹20-50 | Excellent | Very High |
| **Bottle (Glass)** | Sauces, squash, drinks | ₹12-30 | Excellent | High |
| **Bottle (PET)** | Sauces, drinks (budget) | ₹5-12 | Good | Medium |

### Size Strategy

| Size | Purpose | Price Point | Customer |
|------|---------|-------------|----------|
| **Trial (50-100g)** | First purchase, sampling | ₹49-99 | New customer |
| **Regular (200-250g)** | Core SKU | ₹149-249 | Repeat buyer |
| **Family (500g-1kg)** | Value pack | ₹349-599 | Loyal customer |
| **Bulk (1-2kg)** | HoReCa, B2B | ₹499-999 | B2B/power user |

### FSSAI Label Checklist

Every product label MUST include:

- [ ] Product name (in English + one Indian language)
- [ ] Ingredients list (descending by weight)
- [ ] Nutritional information (per 100g)
- [ ] Net quantity (g/ml/kg/L)
- [ ] Manufacturer name & address
- [ ] FSSAI license number + logo
- [ ] Batch/Lot number
- [ ] Manufacturing date + Best before date
- [ ] Veg/Non-veg symbol (green/brown dot)
- [ ] MRP (inclusive of all taxes)
- [ ] Customer care details
- [ ] Allergen declaration (if any)
- [ ] Storage instructions
- [ ] Country of origin

### Visual Design Guidelines (Mom Ka Nuska Style)

| Element | Direction |
|---------|-----------|
| **Colors** | Earthy: terracotta, turmeric yellow, sage green, cream |
| **Typography** | Warm, hand-drawn feel for brand name; clean for info |
| **Imagery** | Mom's hands, farm elements, raw ingredients |
| **Material** | Kraft paper look, matte finish (premium, eco feel) |
| **Logo** | Prominent, recognizable at grocery shelf distance |
| **Differentiator** | "Zero Preservatives" badge prominent |

### Print Specifications

| Spec | Value |
|------|-------|
| **Resolution** | 300 DPI minimum for print |
| **Color mode** | CMYK (not RGB) |
| **Bleed** | 3mm on all sides |
| **File format** | AI/EPS (vector) + high-res PDF |
| **Barcode** | EAN-13 (for retail), QR for D2C tracking |

## Common Mistakes

- ❌ Designing in RGB (prints differently in CMYK)
- ❌ Forgetting FSSAI mandatory label elements
- ❌ Over-designing (too busy = hard to read on shelf)
- ❌ Using non-food-grade packaging material
- ❌ Not testing seal integrity before production run
