---
description: Plan and execute farm setup for food production, testing, and content creation. From land search to operational farm.
---

## Steps

1. **Define Requirements**
   - Use `fmcg-farm-setup` skill
   - Purpose: production + testing + content (triple-duty farm)
   - Land size needed (0.5-2 acres)
   - Budget for setup
   - Proximity to team's base city

2. **Land Search**
   - Define search criteria (distance, water, road, power, internet)
   - Identify 5-10 potential sites
   - Shortlist to top 3 for site visits
   - Output: Property comparison spreadsheet

3. **Site Visits**
   - Visit top 3 sites
   - Check: water, electricity, road, soil, neighbors, security
   - Negotiate lease/purchase terms
   - Gate: Team agrees on 1 site

4. **Infrastructure Planning**
   - Design farm layout (kitchen, studio, garden, storage, parking)
   - Kitchen unit specifications
   - Content studio specifications
   - Get 3 contractor quotes
   - Output: Layout plan + budget

5. **Construction & Setup**
   - Kitchen unit build/renovation
   - Content studio setup
   - Equipment procurement and installation
   - Utility connections (power, water, internet)
   - Kitchen garden planting
   - Gate: All infrastructure functional

6. **Staffing**
   - Hire kitchen helper(s)
   - Hire farm worker
   - Hire videographer (part-time initially)
   - Gate: Core team in place

7. **Operations Launch**
   - First test batch production
   - First content shoot
   - Quality check protocols in place
   - Daily operations schedule established
   - Gate: Farm is producing + creating content
