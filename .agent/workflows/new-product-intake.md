description: Take a new food product from recipe idea to live listing. Use when adding a new SKU to the catalog.
---

## Steps

1. **Product Idea Brief**
   - Category, name, target audience, flavor profile
   - Market gap this product fills
   - Verify it fits brand architecture
   - Gate: Founder approves concept

2. **Recipe Development**
   - CPO (Anjula Singh) develops recipe
   - Minimum 3 iterations with feedback
   - Gate: CPO taste test pass

3. **Recipe SOP**
   - Use `fmcg-product-development` skill
   - Document all ingredients, quantities, process steps
   - Include quality checkpoints
   - Gate: Written SOP complete

4. **Quality Testing**
   - Produce 5 test batches for consistency
   - Start shelf-life testing (accelerated)
   - Gate: All 5 batches taste-consistent

5. **Cost Calculation**
   - Calculate BOM (Bill of Materials)
   - Add processing, packaging, logistics costs
   - Set MRP with >45% gross margin
   - Gate: Margin approved by finance

6. **FSSAI Compliance**
   - Use `fmcg-legal-compliance` skill
   - Determine FSSAI category
   - Prepare nutritional information
   - Verify label requirements
   - Gate: FSSAI-compliant label draft

7. **Packaging**
   - Use `d2c-packaging-design` skill
   - Design label, select material/format
   - Print prototype, seal test
   - Gate: Package approved by branding team

8. **Photography & Content**
   - Product shots (white bg + lifestyle)
   - "Making of" video for social
   - Recipe video featuring the product
   - Gate: Content assets ready

9. **Listing & Launch**
   - List on D2C website
   - List on Amazon/Flipkart
   - Set launch offer (e.g., intro discount)
   - Announce on social channels
   - Gate: Product live and orderable
