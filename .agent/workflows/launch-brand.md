description: End-to-end brand launch pipeline — from market research to first sale. Use when starting a new food brand from scratch.
---

## Steps

1. **Market Research**
   - Use `fmcg-market-intelligence` skill
   - Run TAM/SAM/SOM analysis for the product category
   - Complete SWOT analysis
   - Analyze top 5-8 competitors
   - Output: Market Research Report

2. **Brand Identity**
   - Use `d2c-brand-building` skill
   - Define brand purpose, story, personality, voice
   - Formulate USP
   - Map moat strategy (short/mid/long term)
   - Output: Brand Identity Document

3. **Legal Setup**
   - Use `fmcg-legal-compliance` skill
   - Incorporate company (Pvt Ltd recommended)
   - File trademark application (Class 29 + 30)
   - Register FSSAI (Basic)
   - Register GST
   - Output: Legal Compliance Checklist

4. **Hero Product Development**
   - Use `fmcg-product-development` skill
   - CPO develops recipe, 3+ iterations
   - Write Recipe SOP
   - Shelf-life testing (accelerated)
   - BOM + pricing calculation
   - Output: Recipe SOP + Cost Sheet

5. **Packaging Design**
   - Use `d2c-packaging-design` skill
   - Choose format (jar/pouch/box)
   - Design label with FSSAI compliance
   - Print prototype, review with team
   - Output: Print-ready packaging file

6. **Content & Photography**
   - Use `d2c-content-strategy` skill
   - Product photography (farm + lifestyle)
   - Film first "Farm Diary" content
   - Build 1-month content calendar
   - Output: Photo assets + Content Calendar

7. **Website Setup**
   - Use `d2c-website-strategy` skill
   - Build D2C site (Shopify or custom)
   - Implement location-first UX
   - List on Amazon/Flipkart
   - Output: Live website URL

8. **Launch Campaign**
   - Use `d2c-influencer-marketing` skill
   - Seed product to 50 micro-influencers
   - Launch first referral program
   - Activate first paid ads (₹500/day test)
   - Output: Campaign brief + tracking dashboard

9. **Post-Launch Review (Week 2)**
   - Analyze first 14 days: orders, AOV, CAC, feedback
   - Adjust pricing/offers based on data
   - Double down on what's working
   - Output: Launch Retrospective Report
