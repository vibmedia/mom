### Zone 1: Kitchen / Processing Unit (400 sq ft)

**Purpose:** Core production area — where all products are made

**Layout (Unidirectional Flow):**
```
Wash Station → Prep Area → Cooking Zone → Cooling Area → Exit to Packaging
```

**Equipment:**
| Item | Spec | Est. Cost |
|------|------|-----------|
| Commercial gas stove (2-burner) | SS, heavy duty | ₹15,000 |
| Grinding machine (wet+dry) | 5L capacity | ₹25,000 |
| Mixing vessels (SS 304) | 10L, 20L, 50L | ₹20,000 |
| Work tables (SS 304) | 4x2 ft, 3 units | ₹30,000 |
| Weighing scale (digital) | 0.1g-50kg | ₹5,000 |
| Wash sink (3-compartment) | SS, wall-mounted | ₹15,000 |
| Exhaust hood + chimney | Commercial grade | ₹20,000 |
| Water purifier (RO+UV) | 25L/hr | ₹15,000 |
| Utensils & tools | Various | ₹15,000 |

**FSSAI Compliance:**
- SS 304 food-grade surfaces throughout
- Epoxy-coated flooring with drainage
- Fly catchers, hand wash stations at entry
- Temperature monitoring logs
- Batch tracking register

**Budget: ~₹1.6L equipment + ₹2-3L construction**

---

### Zone 2: Dryer Area (300 sq ft — Open + Covered)

**Purpose:** Sun-drying pickles, papads, spices (critical for authentic taste)

**Design:**
- **Open section (200 sq ft):** Wire mesh drying racks on elevated platforms (2 ft above ground) — for sun-drying papads, amla, chillies
- **Covered section (100 sq ft):** Polycarbonate roof shelter for monsoon drying — keeps rain out, lets UV through
- **Dehydrator backup:** Electric food dehydrator for rainy season consistency

**Equipment:**
| Item | Est. Cost |
|------|-----------|
| SS drying racks (10 units) | ₹15,000 |
| Elevated platform (masonry) | ₹10,000 |
| Polycarbonate roofing | ₹20,000 |
| Food dehydrator (10-tray) | ₹25,000 |
| Mesh covers (insect protection) | ₹5,000 |

**Budget: ~₹75K equipment + ₹50K construction**

---

### Zone 3: Storage (300 sq ft — split into Raw + Finished)

**Purpose:** Separate storage for raw materials and finished goods

**Layout:**
```
┌─────────────────────────────────┐
│  RAW MATERIAL     │  FINISHED   │
│  STORAGE          │  GOODS      │
│  (150 sq ft)      │  (150 sq ft)│
│                   │             │
│  - Spices rack    │  - Labeled  │
│  - Oil drums      │    shelf    │
│  - Dry goods      │  - Ready to │
│  - Vegetables     │    ship     │
│    (temp shelf)   │  - Packaging│
│                   │    material │
└─────────────────────────────────┘
```

**Features:**
- Climate controlled (exhaust fans, no direct sunlight)
- FIFO (First In First Out) rack system
- Pest control: sealed containers, mesh windows
- Inventory board (whiteboard tracking)

**Equipment:**
| Item | Est. Cost |
|------|-----------|
| Industrial shelving (6 units) | ₹18,000 |
| Sealed storage containers (20) | ₹10,000 |
| Exhaust fans (2) | ₹4,000 |
| Pest control setup | ₹3,000 |
| Inventory whiteboard | ₹1,000 |

**Budget: ~₹36K equipment + ₹30K construction**
